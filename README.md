# Next Action plugin for Obsidian

An opinionated GTD-style task management system for Obsidian.

## How to install

1. Install the BRAT plugin from the Community Plugins.
2. Open the Settings page for BRAT.
3. Click "Add Beta Plugin".
4. Paste in this address: `https://github.com/alangrainger/obsidian-next-action`
5. Click "Add Plugin".

## Keyboard shortcuts

Type `?` on the tasklist screen to open the keyboard shortcuts help page.

- `Arrow Up` - Move up the tasklist
- `Arrow Down` - Move down the tasklist
- `j` - Alternative key to move up the tasklist
- `k` - Alternative key to move down the tasklist
- `Escape` - Close the sidebar
- `Enter` - Open the currently active row in the sidebar
- `m` - Move the highlighted task into a project
- `n` - Create a new task. If a project is highlighted, a subtask will be created for that project instead
- `p` - Set the task type to "Project"
- `a` - Set the task type to "Next Action"
- `s` - Set the task type to "Someday"
- `w` - Set the task type to "Waiting On"
- `Space` - Toggle the completed status of the task
